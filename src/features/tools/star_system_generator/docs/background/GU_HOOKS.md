**Geometric Unity: Hook Tables**
**Generator-table source content**
**Companion to GU_GLOSSARY.md**
**Version 1.0**

Source-of-truth for generator rolls beyond the glossary. The glossary's *At the table:* lines anchor each hero term in three play hooks; this file widens that into five table-roll categories — rumors, contracts, encounters, NPCs, and twists — that the generator can sample independently or compose with system rolls.

Each entry is single-sentence, term-tagged for filtering, and written so it can drop into a session without setup. The tag in brackets names the parent glossary term so a system roll that already touches Bleed can preferentially pull `[Bleed]` content for thematic coherence. Roll openly for variety, or filter for focus — both work.

## Using these tables

- **Roll independently or compose with system terms.** A system rolling `Bleed` + `Pinchdrive` can pull a `[Bleed]` rumor and a `[Pinchdrive]` encounter to seed the session.
- **Filter by tag for thematic coherence**, or roll openly for variety. Both work.
- **Entries are intentionally compact** — the GM expands. A rumor is one sentence; the conversation is the GM's.
- **Twists are mid-session, not pre-session.** Don't roll a twist with the contract — roll it after the table has committed to a direction.
- **NPCs are seeds, not sheets.** The seed is what the player remembers; the GM fills in the rest.
